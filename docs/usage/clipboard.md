# Clipboard

Two keys copy to the system clipboard without leaving the browser.

| Key | What it copies |
| --- | -------------- |
| `y` | Absolute path of the highlighted entry |
| `Y` | Full contents of the highlighted file (text files only) |

A confirmation appears in the hint bar after a successful copy: `copied path` or `copied contents (N lines)`.

`Y` is skipped for directories and binary files — it shows a brief notice instead.
