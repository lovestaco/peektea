## Install

**Download a binary** — grab the right archive from the assets below:

| Platform | File |
|----------|------|
| Linux x86-64 | `peektea___VERSION___linux_amd64.tar.gz` |
| Linux arm64 | `peektea___VERSION___linux_arm64.tar.gz` |
| macOS x86-64 | `peektea___VERSION___darwin_amd64.tar.gz` |
| macOS Apple Silicon | `peektea___VERSION___darwin_arm64.tar.gz` |

**With Go:**
```bash
go install github.com/lovestaco/peektea@__VERSION__
```

**From source:**
```bash
git clone https://github.com/lovestaco/peektea
cd peektea && make install
```
