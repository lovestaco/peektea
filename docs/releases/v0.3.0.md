# peektea v0.3.0

> Released 2026-07-01

![cover](IMG:media/cover.png)

## Summary

v0.3.0 adds two new features: a cut/paste move flow (`x` to cut, `v` to paste) so you can move files and directories without dropping to a shell, and clipboard copy (`y` for path, `Y` for contents) so you can grab what you're looking at in one keystroke.

## What's new

### Move files with x / v

Stage any file or directory for moving with `x`, navigate to the destination, then press `v` to drop it there. The hint bar shows what's staged the whole time.

- Name collisions prompt before overwriting (`y/n`)
- Same-directory paste is a no-op
- Moving a directory into itself is refused
- Cross-filesystem moves fall back to copy + delete automatically
- `esc` cancels a staged move

### Copy to clipboard with y / Y

| Key | Action |
|-----|--------|
| `y` | Copy the absolute path of the highlighted entry |
| `Y` | Copy the full contents of the highlighted file |

A confirmation appears in the hint bar after copying. `Y` skips directories and binary files.

## Install

**Download a binary** — grab the right archive from the assets below:

| Platform | File |
|----------|------|
| Linux x86-64 | `peektea_v0.3.0_linux_amd64.tar.gz` |
| Linux arm64 | `peektea_v0.3.0_linux_arm64.tar.gz` |
| macOS x86-64 | `peektea_v0.3.0_darwin_amd64.tar.gz` |
| macOS Apple Silicon | `peektea_v0.3.0_darwin_arm64.tar.gz` |

**With Go:**
```bash
go install github.com/lovestaco/peektea@v0.3.0
```

**From source:**
```bash
git clone https://github.com/lovestaco/peektea
cd peektea && make install
```

## Keys

| Key | Action |
|-----|--------|
| `↑/↓` `k/j` | navigate |
| `→` `l` `enter` | go inside directory |
| `←` `h` `backspace` | go to parent |
| `H` | go to home directory |
| `o` | open with configured program |
| `x` | cut (stage for move) |
| `v` | paste (move staged entry here) |
| `y` | copy absolute path to clipboard |
| `Y` | copy file contents to clipboard |
| `p` | toggle preview panel |
| `[` / `]` | scroll preview |
| `/` | filter as you type |
| `esc` | exit filter / cancel staged move |
| `.` | toggle hidden files |
| `s` | cycle sort: name → size → modified |
| `q` `ctrl+c` | quit |
